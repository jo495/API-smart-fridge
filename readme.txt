API dla inteligentnej lodówki

Umożliwia dodawanie, usuwanie i aktualizowanie stanu produktów spożywczych, dań oraz przepisów w ramach systemu lodówki. Na podstawie analizy cech elementów przechowywanych w systemie, udostępnienia użytkownikowi następujące funkcje:
•	zaproponowania produktów, które powinien w najbliższym czasie kupić, a także które powinien kupić do przygotowania wskazanej potrawy
•	wskazania, które produkty oraz potrawy należy zjeść w ciągu podanej przez użytkownika liczby dni, żeby się nie zepsuły
•	zaproponowanie według określonej strategii (zużywające jak najwięcej produktów, które zepsułyby się w najbliższym czasie, ulubione, wegetariańskie, najzdrowsze), dań, które użytkownik mógłby ugotować ze składników, które posiada 
Ponadto API udostępnia możliwość zmiany dnia poprzez jedno żądanie tak, aby ułatwić zaimplementowanie systemu.
